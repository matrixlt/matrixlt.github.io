微积分拾遗1--数列极限与函数极限

Posted on 2018-10-14

数列极限的直观理解

直观图像

观察定义：

$\lim_\limits{n\to\infty}a_{n} = A \Leftrightarrow \forall \epsilon>0, \exists N, n>N \Rightarrow |a_{n}-A|<\epsilon.$

我们可以画出示意图：

直观的看，数列极限为A的含义就是N之后有连续的无穷多项都在A附近。
这图像是非常有用的，例如考虑以下问题：

设$\lim_\limits{n\to\infty}a_{n} = A, \lim_\limits{n\to\infty}b_{n} = B ,A > B$当n充分大时，是否有$a_{n} > b_{n}$?

类比上面的图像，取$\epsilon < \frac{A-B}{2}$结论是显然的

注意到逆命题是不成立的，如取$a_{n} = \frac{2}{n}, b_{n} = \frac{1}{n}$

子列

如果将自然数集N划分为k个无穷集合$N_{1},N_{1},…,N_{k}$,那么相当于将数列$a_{n}$划分成了k个子列$a_{1n},a_{2n},…,a_{kn}$，有以下的命题成立：

$\lim_\limits{n\to\infty}a_{n} = A \Leftrightarrow \lim_\limits{n\to\infty}a_{1n} =…= \lim_\limits{n\to\infty}a_{kn} = A$

这个命题可以推出极限的唯一性等命题

任意多与无穷

考虑性质$P(n)$对任意$n \in N$成立，那么当n趋于无穷时，$P(n)$并不一定成立。如以下两个命题一真一假

存在任意长的连续合数列，如$(k+1)!+2, (k+1)!+3,…,(k+1)!+(k+1)$
存在无穷长的连续合数列，这显然是错的，因为质数有无穷多个

函数在某点的极限是函数此点附近领域的性质

函数极限的定义和性质可以类比数列极限，这里只强调一点即函数在$x_{0}$处的极限与函数在$x_{0}$的值或事先给定的一定$x_{0}+\epsilon$处的值都无关
这是因为事先给定的一点总不能在考虑极限时包括在$x_{0}$的去心领域内

了解了这一点，加上泰勒公式描述了函数在某点处的多项式展开，利用泰勒公式求某个极限就成为了在函数性质好（满足泰勒定理要求时）的通用方法

利用泰勒公式求极限

首先考察多项式分式的极限
$\lim_\limits{x\to0} \frac{a_{0}+…+a_{n}x^{n}}{b_{0}+…+b_{m}x^{m}}$
我们分别按幂次从低到高考察分子分母的系数，去除所有为零的系数，得到下式
$\lim_\limits{x\to0} \frac{a_{j}x^{j}+o(x^{j})}{b_{k}x^{k}+o(x^{k})}$
那么
$ lim = \frac{a_{j}}{b_{k}} \quad, j = k $
$ lim = 0 \quad, j > k $
$ lim = not \quad exist \quad, j < k $

至此，求极限时我们只需将所求函数展开至足够多项，再应用上面的定理即可
注意到有时已知极限存在，此时只要求出分母的次数与系数，再检查分子大于等于这个次数的项即可

无穷小的观点

无穷小的观点简洁易懂，将极限式化为易于运算的等式，在思考问题时很直观
下面的定理体现了这一点，而且定理本身也是很有用的

定理：设函数$f(x)$在$x_0$处具有直到$k$阶导数，并且
$$f’(x_0)=f’’(x_0)=\cdots=f^{(k-1)}(x_0)=0, f^{(k)}(x_0)\neq0$$
那么:

  • 当$k$为奇数时,$x_0$不是$f(x)$的极值点。
  • 当$k$为偶数时,$x_0$是$f(x)$的极值点。若$f^{(k)}(x_0)<0$则$x_0$是$f(x)$的严格极大值,若$f^{(k)}(x_0)>0$则$x_0$是$f(x)$的严格极小值.

定理的证明
由$$f(x) = f(x_0) + f^{(k)}(x_0)(x-x_0)^k + o((x-x_0)^k)$$
得$$\frac{f(x_0) - f(x_0)}{(x-x_0)^k} = f^{(k)}(x_0) + o(1)$$
故等式左侧在$x_0$附近的符号完全由$f^{(k)}(x_0)$确定.

  • 当$k$为奇数时，$f(x) - f(x_0)$在$x_0$左右两侧变号,$x_0$不是$f(x)$的极值点。
  • 当$k$为偶数时,$f(x) - f(x_0)$在$x_0$左右两侧不变号，若$f^{(k)}(x_0)<0$则$f(x) - f(x_0)<0$,$x_0$是$f(x)$的严格极大值点。同理若$f^{(k)}(x_0)>0$则$f(x) - f(x_0)>0$,$x_0$是$f(x)$的严格极小值点。

例题与解答

一个常见的错误

求$\lim_\limits{x\to0} \frac{1-xcot(x)}{x^{2}cos(x^{2})}$

解1(错误)
$lim = \frac{1-x\frac{cos(x)}{sin(x)}} {x^{2}cos(x^{2})}
= \frac{1-x\frac{1}{sin(x)}} {x^{2}}
= \frac{sin(x)-x}{sin(x)x^{2}}
= \frac{-\frac{x^{3}}{6}+o(x^{3})}{x^{3}+o(x^{3})}
= -\frac{1}{6}$

解2(正确)
$lim = \frac{1-x\frac{cos(x)}{sin(x)}} {x^{2}cos(x^{2})}
= \frac{sin(x)-xcos(x)} {x^{2}cos(x^{2})sin(x)}
= \frac{x-\frac{1}{6}x^{3}+o(x^{3}) -x(1-\frac{1}{2}x^{2}+o(x^{2}))}{x^{3}+o(x^{3})}
= \frac{1}{3}
$

解1错误的原因在于错误的将0带入了cos(x)求值，但解2中带入$cos(x^{2})$是对的，这是因为极限的积等于积的极限(在极限都存在时)
这个例子告诉我们在每一步变换时都要想清楚根据

泰勒公式与无穷小的典型应用

求$\lim_\limits{x\to0} \frac{tan(sin(x))-sin(tan(x))} {tan(x)-sin(x)}$

$tan(x)-sin(x) = sin(x)(\frac{1-cos(x)}{cos(x)}) = \frac{x^{3}}{2} + o(x^{3})$

通过定义求tan(x)的泰勒展开得$tan(x) = x + \frac{x^{3}}{3} + o(x^{3})$

$tan(sin(x))-sin(tan(x)) = sin(x)-\frac{sin^{3}(x)}{3}+o(sin^{3}(x)) -(tan(x)-\frac{tan^{3}(x)}{6}+o(tan^{3}(x)))$
$ = x - \frac{x^{3}}{6} + \frac{x^{3}}{3} -(x+\frac{x^{3}}{3}-\frac{x^{3}}{6}) + o(x^{3})
= 0 + o(x^{3})$

故极限为0

求$\lim_\limits{n\to\infty} n[(1+\frac{x}{n})^{n}-e^{x}]$
